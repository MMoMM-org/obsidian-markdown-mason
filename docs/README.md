# Documentation

Documentation for **Markdown Mason**, a desktop Obsidian plugin that reshapes pasted or
selected text to fit the structure of your note — cascading heading levels and filing
citations as deduplicated footnotes. This is the docs index; start with
[Installation](installation.md) or the [Usage](usage.md) walkthrough.

## Overview

Markdown Mason is for Obsidian users who paste answers from Perplexity or other LLM/web
sources into existing notes and want them to fit in: headings cascaded relative to the
cursor, inline citations turned into real footnotes, and those footnotes renumbered and
deduplicated against what's already in the note. The transforms run as small, consent-gated
**scripts** managed by the plugin — installed from a curated library or imported from your
vault — rather than loose files scattered through your vault. It is desktop-only and requires
Obsidian 1.6.6 or newer.
